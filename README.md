# Risk and Return Analysis in Equities

## Overview

This repository presents my MBA Finance research project titled **"Risk and Return Analysis in Equities."** The project analyzes the relationship between investment risk and expected returns using financial and statistical techniques on selected equity stocks.

---

## Project Objectives

- Analyze the risk and return of selected equity stocks.
- Compare the performance of different companies.
- Evaluate investment decisions using financial metrics.
- Apply statistical techniques for equity analysis.
